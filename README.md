# Church CBT Exam System

A Computer-Based Test (CBT) web application built for church use, designed to conduct timed multiple-choice examinations.

## Features
- Candidate name input
- Timed examination
- Multiple-choice questions
- One question per page navigation
- Automatic submission on completion
- Confirmation message after submission

## Technologies Used
- HTML5
- CSS3
- JavaScript (Beginner level)

## Purpose
This project was built as a practical solution for church examinations and to improve my JavaScript and DOM manipulation skills.
